1DevOps — host on forge.sashai.tech P1 · 30 min

  1. Create CNAME file in repo root containing forge.sashai.tech (single line, no protocol). Commit & push to main.
  2. DNS (at registrar of sashai.tech) — add one record: 
    Type: CNAME   Name: forge   Value: sovitgarg.github.io.   TTL: 300
    If root-domain redirect is wanted later, use four A records pointing to 185.199.108.153, .109.153, .110.153, .111.153.
  3. GitHub: Settings → Pages → enter forge.sashai.tech as Custom domain. Wait for DNS check (~5 min). Tick Enforce HTTPS.
  4. Rewrite canonical URLs from sovitgarg.github.io/sash_ai_forge → forge.sashai.tech in: index.html (canonical, OG, Twitter, 6× JSON-LD), llms.txt, sitemap.xml, robots.txt, 404.html, assets/js/config.js. One grep: grep -rl "sovitgarg.github.io/sash_ai_forge" .
  5. Submit to Google Search Console (new property: forge.sashai.tech) + Bing Webmaster Tools. Submit sitemap.
  6. Verify: curl -I https://forge.sashai.tech/ returns 200 · valid TLS cert · no redirect chain · Cache-Control present.
Rollback plan: if DNS goes wrong, remove the CNAME record and the repo's CNAME file — site falls back to sovitgarg.github.io/sash_ai_forge/ automatically, zero downtime.
